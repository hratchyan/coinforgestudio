# Keyboard Shortcuts

## Files & app
| Keys | Action |
| --- | --- |
| Ctrl+N | New coin |
| Ctrl+O | Projects browser |
| Ctrl+S | Save project |
| Ctrl+Shift+S | Save as new copy |
| Ctrl+E | Export for engraving |
| Ctrl+B | Background Remover utility |
| Ctrl+L | Generate cut outline |
| F1 | Help |

## Editing
| Keys | Action |
| --- | --- |
| Ctrl+Z / Ctrl+Y | Undo / Redo |
| Ctrl+D | Duplicate selection |
| Delete / Backspace | Delete selection |
| Ctrl+A | Select all |
| Esc | Deselect |
| Ctrl+V | Paste image from clipboard (screenshots!) |
| Double-click text | Edit text in place |

## Moving & transforming
| Keys | Action |
| --- | --- |
| Arrow keys | Nudge 0.2 mm |
| Shift + Arrows | Nudge 1 mm |
| [ / ] | Rotate −1° / +1° (rings: slide around the circle) |
| Shift + [ / ] | Rotate 15° steps |
| Shift while dragging | Constrain to one axis |
| Shift while rotating | Snap to 15° |

## View
| Keys | Action |
| --- | --- |
| Scroll wheel | Zoom at cursor |
| Space + drag / Alt + drag / middle-drag | Pan |
| Drag empty space | Pan |
| F | Fit coin to window |
| G | Toggle guides |

## On the canvas
| Gesture | Action |
| --- | --- |
| Click | Select |
| Shift+click | Add/remove from selection |
| Drag body | Move (rings: change radius) |
| Corner squares | Scale |
| Yellow handle | Rotate / slide around ring |
| Green diamond | Ring radius |
| Drag & drop image file | Smart Import |
| Drag & drop .coin file | Open project |
